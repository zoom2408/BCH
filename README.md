# 📚 Bạn Cùng Học

> Nền tảng e-learning cho sinh viên **Dược học** và **Tâm lý học** — học thông minh hơn, nhớ lâu hơn.

---

## ✨ Tính năng nổi bật

| Tính năng | Mô tả |
|---|---|
| 📖 **Bài viết** | Đọc nhanh kiến thức theo chủ đề |
| 🎓 **Khóa học** | Bài học có nội dung, quiz và chứng chỉ |
| 💡 **Key Essence** | Lưu tinh hoa bài học vào sổ tay cá nhân |
| 🧪 **Quiz bài học** | Kiểm tra ngay cuối mỗi bài — không tính điểm, chỉ để ôn |
| 📝 **Kiểm tra cuối khóa** | Đạt ≥80% → nhận chứng chỉ hoàn thành |
| 📋 **Sổ tay** | Xem lại tất cả essence đã lưu từ mọi bài học |

## 🏗️ Tech Stack

Bản MVP được viết hoàn toàn bằng HTML/CSS/JS thuần — **zero dependencies**, **zero build step**.

| Layer | Công nghệ |
|---|---|
| Markup | HTML5 |
| Style | CSS custom properties (design tokens) |
| Logic | Vanilla JavaScript (ES6+) |
| Fonts | Google Fonts — Lora, DM Sans, DM Mono |
| Icons | Unicode emoji |

## 🎨 Design System

- **Màu nền:** Cream ấm `#f7f4ef`
- **Typography:** Lora (tiêu đề serif) + DM Sans (body) + DM Mono (số/code)
- **Accent:** Xanh lá (Dược) · Xanh dương (Tâm lý) · Tím (Premium)
- **Border radius:** 10px · **Shadow:** subtle `0 2px 12px rgba(26,24,20,.07)`
